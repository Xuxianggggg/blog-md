IEEE 754 单精度浮点数转换器

输入一个十进制数，查看它在 32 位 float 中的 sign、exponent、fraction，以及规格数、非规格数、Infinity、NaN 的判断过程。

支持：8.6875、-0.1、1e-40、2^-127、Infinity、NaN
8.6875 正常数 2^-127 非规格数 2^-149 最小正非规格数 16777217 精度丢失 0.1 舍入 Infinity NaN
法语关键词：规格数 = nombre normalisé；非规格数 = nombre sous-normal；尾数 = mantisse / fraction；指数 = exposant。
