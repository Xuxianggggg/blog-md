# 通过树莓派 5 远程开机笔记本

## 目标

使用树莓派 5 作为家里常开的设备，通过 Wake-on-LAN 远程唤醒笔记本电脑。

基本结构：

```text
远程设备
  ↓ SSH / Tailscale
树莓派 5
  ↓ Wake-on-LAN 魔术包
笔记本电脑开机
```

---

## 前提条件

- 树莓派 5 和笔记本在同一个局域网内。
- 笔记本使用有线网卡连接网络。
- 笔记本插着电源。
- 笔记本的有线网卡支持 Wake-on-LAN。
- Windows 网卡设置中已开启网络唤醒。

---

## Windows 端设置

在 Windows 的设备管理器中找到有线网卡，例如：

```text
Realtek PCIe GbE Family Controller
```

进入：

```text
设备管理器
→ 网络适配器
→ 有线网卡
→ 属性
```

### 高级设置

建议确认以下选项已开启：

```text
魔术封包唤醒：开启
关机网络唤醒：开启
```

不同电脑上的名称可能略有不同，例如：

```text
Wake on Magic Packet
Shutdown Wake-On-LAN
Wake from S5
```

### 电源管理设置

建议勾选：

```text
允许计算机关闭此设备以节约电源
允许此设备唤醒计算机
只允许幻数据包唤醒计算机
```

---

## 查询笔记本有线网卡 MAC 地址

在 Windows 中打开 CMD 或 PowerShell：

```powershell
ipconfig /all
```

找到有线网卡对应的：

```text
物理地址 / Physical Address
```

例如格式可能是：

```text
AA-BB-CC-DD-EE-FF
```

在树莓派命令中通常写成：

```text
AA:BB:CC:DD:EE:FF
```

---

## 树莓派端安装 Wake-on-LAN 工具

在树莓派上执行：

```bash
sudo apt update
sudo apt install wakeonlan
```

---

## 远程开机命令

在树莓派上执行：

```bash
wakeonlan AA:BB:CC:DD:EE:FF
```

把 `AA:BB:CC:DD:EE:FF` 换成笔记本有线网卡的 MAC 地址。

---

## 推荐测试方式

先测试睡眠唤醒：

```text
1. 笔记本插电源并接网线
2. 让笔记本进入睡眠
3. 在树莓派上执行 wakeonlan 命令
4. 检查笔记本是否被唤醒
```

如果睡眠唤醒成功，再测试完全关机唤醒：

```text
1. 笔记本正常关机
2. 等待几秒钟
3. 在树莓派上执行 wakeonlan 命令
4. 检查笔记本是否开机
```

---

## 常见问题

如果睡眠可以唤醒，但关机后不能唤醒，可以尝试在 Windows 管理员 PowerShell 中关闭快速启动：

```powershell
powercfg /h off
```

然后重新测试关机唤醒。

Wake-on-LAN 对有线网卡更稳定。Wi-Fi 的网络唤醒通常只适合睡眠状态，不一定支持完全关机后开机。

---

## 最终使用方式

只要树莓派可以远程访问，就可以通过树莓派执行：

```bash
wakeonlan AA:BB:CC:DD:EE:FF
```

笔记本开机后，再使用远程桌面、Tailscale、VNC、RustDesk 或其他远程控制工具连接笔记本。
